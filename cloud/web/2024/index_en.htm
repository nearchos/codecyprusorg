Theme

The theme of Code Cyprus 2024 is Coding for All.

This year, Code Cyprus welcomes a keynote from the co-founders of one of the most successful high-tech startups in Cyprus. Founded by Maria, Artemis, and Liza in 2020, Malloc helps people, organisations, and enterprises to detect spyware, secure their data and regain their privacy.

The three of them will share their story on how they each got into coding, and how that led them to forming a very popular software company! They will sure get you inspired to start thinking of the bright future that lies ahead.

Workshops

Select one of the workshops, and receive a cool Certificate of Participation while also learning to code!

These will be running in parallel, thus each participant must select exactly one workshop.

Build your own 3D T. Rex Game  Computer lab 014 on the ground floor

This workshop teaches simple coding principles guiding you through a multimedia-rich experience with 3-D animation. Participants use basic programming skills to build their own Dino Run game. NOVICE
With Dr Josephina Antoniou and Dr Louis Nisiotis

Programming a Robotic Arm Electronics lab 020 on the ground floor

Robotics and electronic are among the most interesting applications of programming. In this workshop you will learn how to code a robotic arm that simulates a human arm, and teach it how to execute simple tasks, such as moving an object. You will also learn how to code an Arduino board to control the movements of a robotic system. INTERMEDIATE
With Dr Marios Raspopoulos and Dr Stelios Ioannou and Dr Panagiota Nikolaou

Drive your own rover on Mars Computer lab 114 on the 1st floor
How do you tele-control a robotic vehicle on Mars some 50 million kilometers from our planet earth? The answer is coding. Learn how these robotic explorers can be programmed with a sequence of commands which are communicated and executed on the vehicle to achieve a specific goal. Using a graphical programming language and your own creativity, send your code to a real robot and watch it as it explores the unknown landscape on Mars and complete your mission. INTERMEDIATE
With Dr Andriani Piki and Dr Cand. Christiana Varda

Harnessing the power of data analytics  Computer lab 113 on the 1st floor

This workshop introduces an overview of data analytics and its main techniques, and shows how these can be used to discover important information in raw data. The workshop provides an introduction to Python for data analysis and is appropriately adjusted for senior high school students. ADVANCED
With Dr Panayiotis Andreou

Treasure Hunt

Learning code is not only fun, it can also be exciting. This is why we are organizing this Treasure Hunt!

Play alone or with a team-mate. Download the Code Cyprus Android App or choose the HTML5 Web App and follow the clues that lead to the treasure.

The first six winners will get cool prizes!


A €50 coupon for purchases at Public Cyprus for each of the first 6 winners...

Please note that every person can participate in exactly one team, and only one time.

Frequently Asked Questions

What is Code Cyprus?

Code Cyprus is an annual event that aims to inspire the next generation of computer scientists. The main goal of Code Cyprus is to inspire high school students to take an interest in coding. You can learn more about the story of Code Cyprus by reading this page and this blog post. Also, you can read stories and view pictures from previous years of Code Cyprus in our page on facebook.

What should I know before my visit at UCLan Cyprus?

Code Cyprus takes place at one of the most modern university campuses in Europe: UCLan Cyprus. Before your visit, please make sure to learn about the signs and processes of the university by reading the health and safety guide.

Is there a participation cost?

Since its inception, Code Cyprus has been free to its participants, thanks to kind sponsorships from various partners. Our aim is to keep the event free every year, and we will work hard to achieve this. Please note that despite this, we always require a registration (which is free) because we have a limited number of seats.

In what language is the event delivered?

Our aim is to keep the event as bilingual as possible: anyone who understands Greek or English should be able to take the most out of the event.

What do I need to participate to the Treasure Hunt?

You need a mobile device (smartphone or tablet). Ideally you will have a modern Android device to use our native app. If you only have another device (e.g. iPhone, iPad, Windows phone, etc.) you can use our Web App. In any case, make sure you activate your GPS.

Does this include lunch?

Since Code Cyprus starts in the morning and ends in the early afternoon, we have a lunch break. This break is commonly offered by our hosting partner and co-organizer, UCLan Cyprus

Can I sponsor this event?

Yes. Our main goal is to deliver a highly engaging event for the benefit of our participants. Since we aim to keep this free to participating students, we always welcome kind sponsorships. If you are interested, please email us at: hello@codecyprus.org.

Who should I contact if I have another question?

If you have a question, please email us at: hello@codecyprus.org. We generally reply within 24 hours.
